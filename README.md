# Book Tracker

Мини-приложение на Flask для учета прочитанных и запланированных книг.

## Возможности
- список книг
- добавление, редактирование и удаление
- карточка книги
- поиск по названию, автору и жанру
- фильтрация по статусу
- SQLite-хранилище
- pytest-тесты

## Запуск
```bash
pip install -r requirements.txt
python app.py
```

## Тесты
```bash
pytest
```
