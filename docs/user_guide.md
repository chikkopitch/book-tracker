# Руководство пользователя

## Запуск
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить приложение: `python app.py`
3. Открыть `http://127.0.0.1:5000`

## Как пользоваться
1. На главной странице посмотреть список книг.
2. Нажать «Добавить книгу».
3. Заполнить форму и сохранить запись.
4. Открыть карточку книги для просмотра деталей.
5. Использовать «Редактировать» для изменения данных.
6. Использовать поиск и фильтр в верхней части списка.

## Формат данных
- Название, автор и жанр обязательны.
- Количество страниц должно быть больше 0.
- Оценка задаётся от 1 до 5 или не указывается.
